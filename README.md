# Favour Guest Homes — Yaoundé, Cameroon

A modern website for Favour Guest Homes, a guest house located at Carrefour Etougébé, Yaoundé, Cameroon.

## Features

- 🏠 Room management with availability tracking
- ⭐ Guest reviews and ratings
- 🗺️ Interactive directions with OpenStreetMap
- 📹 Video directions from Station Petrolex Etougébé
- 📱 WhatsApp reservation system

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Framer Motion for animations
- Leaflet + OpenStreetMap for maps
